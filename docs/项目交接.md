## 六、已掌握的知识点

### 环境与工具
- Python 多版本共存，PATH 决定调用哪个
- pip 包管理
- VS Code 跑 Python（输出在终端，不弹独立窗口）
- WizTree 诊断磁盘空间
- Steam 库迁移

### AI API 核心概念
- API = 通过网络叫一份 AI 回答
- API Key = 身份证 + 银行卡
- token = 计费单位，输入输出都计费
- Endpoint / base_url
- DeepSeek 兼容 OpenAI 接口规范

### messages 结构
- 列表装多条消息（多轮对话）
- 字典装一条消息（role + content）
- 三种 role：
  - `system` - 设定 AI 身份和规则
  - `user` - 用户提问
  - `assistant` - AI 之前说过的话（用于伪造历史实现"记忆"）
- AI 本身无记忆，"记忆"靠程序传历史

### Python 语法
- `import` 两种方式（`import X` vs `from X import Y`）
- 类与对象：`OpenAI(...)` 是创建对象，内部调用 `__init__`
- 关键字参数 vs 位置参数
- 点访问：`.` = 进入下一层，`()` = 执行
- `response.choices[0].message.content` 套娃式访问
- `input("提示语：")`
- `while True` + `break` + `continue`
- 字符串比较 `==`、`.lower()`、`.isascii()`、`.isalpha()`
- 德摩根定律：`not (A and B)`
- f-string：`f"...{var}..."`
- 三引号多行字符串
- 引号嵌套规则

### 工程化实践
- 配置和代码分离（`.env` + `os.getenv`）
- `.gitignore` 防泄露（注释只支持行首 `#`）
- 永不 print 敏感信息
- 输入校验先于业务处理
- 改动前先备份
- 读 IDE 视觉反馈（颜色一致 = 没拼错）

## 八、当前 main.py 代码

```python
import os
from dotenv import load_dotenv
from openai import OpenAI

load_dotenv()
api_key = os.getenv("DEEPSEEK_API_KEY")

client = OpenAI(
    api_key=api_key,
    base_url="https://api.deepseek.com"
)

while True:
    user_input = input("请输入要学习的英文单词:")

    if user_input.lower() == "quit":
        break

    if not (user_input.isascii() and user_input.isalpha()):
        print("请输入英语单词！")
        print()
        continue

    response = client.chat.completions.create(
        model="deepseek-chat",
        messages=[
            {"role": "system",
             "content": """你是一位英语老师。对每个单词给出：
                    中文释义、词性、英文例句、例句中文翻译
                    如果不是真实英文单词，只回复"无此单词"。"""},
            {"role": "user", "content": f"请讲解单词：{user_input}"}
        ]
    )

    print("AI 回答：", response.choices[0].message.content)
    print()
```
